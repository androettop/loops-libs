# Text input

Textinput is a text input component, can be used to construct forms or other
components that require a text input.

If you want to create a form see the Form/Fields components.

## Basic text input

You can set a small label in top of the text input with the **label** prop.
You can also specify a type and a placeholder like in a HTML standard input.

## Multiline (Text Area)

The text input component can be used as a HTML textarea setting the **multiLine** prop.
You can also use all the native text area props.

## Validations

You can validate the fields and show the errors with the props **touched** and **error**.

The error message will be shown in the input only if the field is touched.
