# Toggle Switch

This component works the same way as a Checkbox (in fact it is internally a Checkbox).

## Sample toggle switch

You can set the prop **value** to true or false to change its value and get its value
with the callback **onChange**.

## Validations

You can validate the toggle switch and show the errors with the props **touched** and **error**.

The error message will be shown in the input only if the field is touched.
