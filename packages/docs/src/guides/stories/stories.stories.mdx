# Story creation

<LoopsMeta readingTime={5} author="Pablo Androetto" />

### Introduction

This guide will help you to understand how to create good stories for your componentes.

### Story generation

You can generate a new story running the following command in your terminal

```
$ ./yarn generate
```

A wizard will guiate to the story creation process.

### Component story structure

The component story will contain a short description, a Figma frame, and a set of examples and technical explanations (if required).

If the component has a visual representation you should link it to figma, for this you can enable the [NodeInspector plugin](https://www.figma.com/community/plugin/758276196886757462) and copy the NodeID of the component group.

![Node inspector](https://i.imgur.com/Z29tYgE.jpg)

![Copying the node id](https://i.imgur.com/vHccCnO.jpg)

Later you can set the node in the FigmaFrame

```jsx
## Figma

<FigmaFrame nodeId="785:6155" />
```
